# efb-qq-plugin-go-cqhttp-docker

A little docker for efb-qq

## How to Use?
1. git clone this repo
2. Change files 
   ```
   efb mods           - requirement/efb_package.txt
   efb mods configs   - config folder
   go-cqhttp configs  - cqhttp_data folder
   ``` 
3. Run docker
   `docker-compose up -d`
